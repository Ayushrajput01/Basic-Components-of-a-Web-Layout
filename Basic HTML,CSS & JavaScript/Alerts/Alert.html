Todo List

  • "new" - Add A Todo
  • "list" - List All Todos
  • "delete" - Remove Specific Todo
  • "quit" - Quit App

RULES :

  • You should enter a specific keyword of the game.
  • Your entered keyword is not specific means, It will Shows the 'Alert' message.
  • If you enter a list keyword, it will shows the 'Information' message.
  • If you enter a delete keyword, it will shows the 'Warning' message.
  • If you enter a quit keyword, it will exits you from the game and it will shows the 'Success' message.
